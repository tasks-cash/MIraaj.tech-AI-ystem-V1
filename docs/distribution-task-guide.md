# Distribution tasks

Prompt 6 turns an approved Prompt 4 campaign revision into a reviewed distribution template and reviewed audience copy. Only approved templates and copy variants can create assignments. Each assignment receives a random token hash, opaque tracked URL, proof marker, locally rendered QR, and locally rendered branded header. Assets remain in private MinIO and API responses contain only short-lived signed reads.

The dentist acceptance profile targets an Arabic (`ar-DZ`, RTL) professional Facebook community in Algeria. It must reuse approved campaign claims, require a QR, marker, disclosure and screenshot, and route private-group proof to human review.

There is no automatic social posting, social OAuth, browser automation, private-group scraping, or reward crediting.
